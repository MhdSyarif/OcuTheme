== Installation ==
Manual installation:

1. Upload the `ocutheme` folder to the `/wp-content/themes/` directory

Activiation and Use

1. Activate the Theme through the 'Themes' menu in WordPress
2. See Appearance -> ocutheme` Options to change theme specific options
